# AEGIS v3.1 고급 매매 기법 문서

---

## 1. 스캘핑 (Scalping) - 단기 초단타 매매
**목표 수익률:** 1-3% (포지션 규모 대비)

### 진입 조건
- **거래량 급증:** 전일 평균 대비 150% 이상 거래량 발생 시
- **호가 스프레드 분석:** 평균 스프레드 대비 30% 이내로 좁혀진 상태
- **시간대:** 시초가(09:00~09:30) 또는 장종가(15:00~15:30) 집중

### 청산 조건
- **목표 도달:** 1-3% 수익 시 즉시 청산
- **시간 청산:** 30분 내 목표 미달성 시 무조건 청산
- **손절:** -1% 하락 시 즉시 매도

### AEGIS 점수 기준
```
호가 밀집도: 8점 이상
주문 유동성: 9점 이상
단기 변동성: 7점 이하 (낮을수록 유리)
시장 심리: 6점 이상 (안정적일 때)
총점: 30점 이상 시 진입 권장
```

### 리스크 관리
- 최대 포지션: 총 자본의 15% 이내
- 일일 최대 손실: -2% (전일 대비 총 자본)
- 동시 포지션: 3개 종목 이내

### 백테스트 결과 (2022-2023)
- **평균 수익률:** 1.8% (진입 당)
- **승률:** 72.3%
- **샤프 비율:** 1.45
- **최대 연속 손실:** 4회

### 주의사항
- 수수료와 세금을 반드시 고려할 것
- 초단타 매매는 심리적 소모가 큼
- 고빈도 매매 시 시스템 장애 대비 필수

## 2. 스윙 트레이딩 (Swing Trading) - 중기 추세 매매
**목표 수익률:** 5-15%

### 진입 조건
- **추세 확인:** 20일 EMA > 50일 EMA (상승 추세)
- **기술적 지표 조합:**
  - RSI(14): 40~60 구간 (중립 → 상승 전환)
  - MACD: 신호선 크로스업 + 히스토그램 양전환
  - 볼린저밴드: 밴드 중앙선 이상 위치, 밴드 수렴 상태
- **형태학적 패턴:** 플래그, 페넌트, 컵 앤 핸들

### 청산 조건
- **익절:** 목표 수익률 5-15% 도달 시
- **추적 손절:** 5일 이동평균선 이탈 시
- **시간 청산:** 5~10영업일 경과 시 재평가

### AEGIS 점수 기준
```
추세 강도: 7점 이상
지표 일치도: 8점 이상
거래량 확인: 7점 이상
시장 환경: 6점 이상 (상승장/횡보장)
총점: 28점 이상 시 진입 권장
```

### 리스크 관리
- 포지션 크기: 총 자본의 10-20%
- 최초 손절: -5% (트레일링 스탑 적용 전)
- 리스크/보상 비율: 1:2 이상

### 백테스트 결과 (2020-2023)
- **평균 수익률:** 9.2% (진입 당)
- **승률:** 65.8%
- **평균 보유 기간:** 7.2일
- **최대 낙폭:** -8.3%

### 주의사항
- 추세 반전 신호에 민감하게 반응할 것
- 횡보장에서는 진입 자제
- 이익 실현 구간 사전 설정 필수

## 3. 갭 트레이딩 (Gap Trading) - 시가 갭 활용 매매
**갭 유형:** 갭상승(Gap Up), 갭하락(Gap Down)

### 진입 조건
**갭상승 시 (Gap Up):**
- 전일 고가 대비 2% 이상 시가 상승
- 거래량 확인: 전일 대비 120% 이상
- 지지선 돌파 확인

**갭하락 시 (Gap Down):**
- 전일 저가 대비 2% 이상 시가 하락
- 저항선 붕괴 확인
- 약세 지표 일치(RSI >70 → 하락)

### 청산 조건
**갭 메우기 시나리오:**
- 갭의 70% 회복 시 부분 청산
- 갭 완전 회복 시 전량 청산

**갭 지속 시나리오:**
- 목표가: 갭 크기의 150% 지점
- 손절: 갭 반대 방향으로 50% 이동 시

### AEGIS 점수 기준
```
갭 크기 점수: 6점 이상 (2-5% 이상)
거래량 점수: 8점 이상
시장 환경 점수: 7점 이상
총점: 21점 이상 시 진입 권장
```

### 리스크 관리
- 포지션 크기: 총 자본의 8-12%
- 손절: 갭의 50% 반대 방향 이동 시
- 최대 손실: 총 자본의 -1% 이내

### 백테스트 결과 (2021-2023)
- **갭 메우기 전략 승률:** 68.4%
- **갭 지속 전략 승률:** 61.2%
- **평균 수익률:** 4.7%
- **최대 손실:** -3.8%

### 주의사항
- 시초가 30분 내 변동성 극대 → 진입 시기 중요
- 뉴스성 갭과 기술적 갭 구분 필수
- 갭 크기가 너무 크면(>10%) 회복 가능성 낮음

## 4. 눌림목 매수 (Pullback Buying) - 추세 조정 구간 매매

### 진입 조건
- **상승 추세 확인:** 고점/저점 상승 중
- **조정 구간 식별:**
  - 20일 EMA 지지 테스트
  - 피보나치 38.2%~61.8% 되돌림 구간
  - 주요 지지선(전 고점) 테스트
- **반전 신호:** 망치형, 역망치형 캔들

### 청산 조건
- **익절:** 이전 고점 돌파 시 또는 목표 수익률 8-12%
- **손절:** 지지선 아래 -3% 하락 시
- **부분 청산:** 저항선 테스트 시 50% 청산

### AEGIS 점수 기준
```
추세 강도: 8점 이상
조정 깊이: 7점 이상 (적절한 되돌림)
반전 신호: 6점 이상
거래량 패턴: 7점 이상 (조정 시 거래량 감소)
총점: 28점 이상 시 진입 권장
```

### 리스크 관리
- 진입 가격: 지지선 근처에서 분할 매수
- 손절: 지지선 아래 -3%
- 리스크/보상: 1:2.5 이상

### 백테스트 결과 (2019-2023)
- **평균 수익률:** 10.5%
- **승률:** 69.1%
- **평균 보유 기간:** 15.3일
- **최대 조정 깊이:** -12.4% (회복 후 수익)

### 주의사항
- 단기 조정 vs 추세 반전 구분 중요
- 거래량 감소 패턴 확인 필수
- 지나치게 깊은 조정(>61.8%)은 추세 약화 신호

## 5. 돌파 매매 (Breakout Trading) - 저항선 돌파 매매

### 진입 조건
- **저항선 확인:** 3회 이상 테스트된 명확한 저항선
- **돌파 신호:**
  - 저항선 상단 1% 이상 이탈
  - 거래량 180% 이상 급증
  - 큰 양봉 발생
- **시장 환경:** 상승장 또는 횡보장 돌파

### 청산 조건
- **익절:**
  - 목표가: 저항선에서 지지선까지 폭의 1:1 이동
  - 또는 10-20% 수익률 도달
- **손절:**
  - 거짓 돌파 시: 저항선 아래로 재진입 시
  - 기계적 손절: -4%

### AEGIS 점수 기준
```
저항 강도: 7점 이상 (명확한 저항선)
돌파 확증: 9점 이상 (거래량 + 캔들)
시장 환경: 6점 이상
이격도: 5점 이상 (과매수/과매도 아님)
총점: 27점 이상 시 진입 권장
```

### 리스크 관리
- 포지션: 총 자본의 10-15%
- 초기 손절: 돌파 지점 -3%
- 거짓 돌파 대비 스탑로스 필수

### 백테스트 결과 (2020-2023)
- **진짜 돌파 승률:** 63.5%
- **거짓 돌파 비율:** 36.5%
- **평균 수익률:** 12.8%
- **손실 제한:** 평균 -3.2%

### 주의사항
- 거짓 돌파(False Breakout) 빈발 → 확증 필수
- 거래량 미확인 돌파는 위험
- 과매수 상태(RSI >70)에서 돌파는 신뢰도 낮음

## 6. 분할 매수/매도 (Scale In/Out) - 포지션 관리 전략

### 진입 계획 (3단계)
1. **1차 진입 (33%):** 초기 신호 발생 시
   - 기본 기술적 조건 충족 시
   - AEGIS 점수 25점 이상

2. **2차 진입 (33%):** 예측 방향 확인 시
   - 가격 2-3% 유리한 방향 이동
   - 거래량 증가 확인
   - 추가 지표 확인

3. **3차 진입 (34%):** 추세 강화 시
   - 주요 저항/지지 돌파
   - 목표 방향으로 명확한 이동
   - 시장 심리 긍정적 전환

### 청산 계획 (3단계)
1. **1차 청산 (50%):** 부분 익절
   - 목표 수익률의 50% 도달 시
   - 단기 저항/지지 테스트 시

2. **2차 청산 (30%):** 추적 익절
   - 목표 수익률 80% 도달 시
   - 기술적 과매수/과매도 신호 시

3. **3차 청산 (20%):** 최종 청산
   - 최종 목표가 도달
   - 추세 반전 신호 발생 시

### AEGIS 점수 기준
```
단계별 평가:
1차: 25점 이상
2차: 추가 5점 이상 (총 30점)
3차: 추가 5점 이상 (총 35점)
```

### 리스크 관리
- 총 포지션: 분할 후 총 자본의 20% 이내
- 평균 단가 관리: 각 단계 간 2-3% 이상 간격
- 최대 손실: 총 투자금의 -5% 한도

### 백테스트 결과 (2018-2023)
- **분할 매수 효과:** 평균 매수가 -2.1% 개선
- **분할 매도 효과:** 평균 매도가 +1.8% 개선
- **전략 승률:** 67.3% → 71.2% (일괄 매수 대비)
- **샤프 비율:** 1.32 → 1.51 개선

### 주의사항
- 모든 자본을 동일 가격에 진입하지 말 것
- 시장 상황에 따라 단계 조정 가능
- 심리적 안정에 유리하나 기회비용 존재

## 7. 손절/익절 규칙 (Risk Management Rules)

### 기계적 손절 규칙
1. **고정 비율 손절:**
   - 단기: -2% (스캘핑)
   - 중기: -3% ~ -5% (스윙/돌파)
   - 장기: -7% (투자)

2. **기술적 손절:**
   - 지지선/저항선 이탈 시
   - 이동평균선 이탈 (중요 선)
   - 패턴 실패 시

3. **시간 손절:**
   - 예상 보유 시간 150% 초과 시
   - 수익/손실 없이 횡보 시

### 익절 규칙
1. **고정 목표익:**
   - 리스크 대비 보상비율 1:2 이상
   - 최소 수익률 3% 이상

2. **추적 손절 (트레일링 스탑):**
   - 고점 대비 -2% 하락 시 청산
   - 이동평균선 이탈 시
   - ATR(14) × 1.5 사용

3. **조건부 익절:**
   - 저항선 도달 시 부분 청산
   - 기술적 과매수 신호 시

### AEGIS 점수 기반 조정
```
시장 변동성 (VIX):
- 낮음(<15): 손절폭 축소 (-0.5%)
- 중간(15-25): 기본 규칙 적용
- 높음(>25): 손절폭 확대 (+1%)

AEGIS 총점:
- 35점 이상: 목표익 확대 (+20%)
- 25점 미만: 손절폭 축소 (-1%)
```

### 백테스트 결과 (2019-2023)
- **손절 효과:** 최대손실 -48% → -22% 개선
- **트레일링 스탑:** 수익률 12% → 15% 증가
- **승률 변화:** 58% → 66% 개선
- **샤프 비율:** 0.85 → 1.42 개선

### 주의사항
- 감정적 손절/익절 절대 금지
- 한 번 설정한 규칙은 철저히 준수
- 시장 환경에 따라 규칙 정기 재검토
- 모든 매매에 손절가 미리 설정

## 종합 운용 가이드라인

### 포트폴리오 구성
- 동시 보유 종목: 3-5개
- 전략 분산: 단기(30%), 중기(50%), 특수(20%)
- 섹터 분산: 동일 섹터 40% 초과 금지

### 일일 루틴
1. **전날 준비 (21:00):**
   - 갭 예상 종목 분석
   - 주요 지표 업데이트

2. **장전 분석 (08:30):**
   - 오늘의 전략 선정
   - 관심 종목 리스트 업데이트

3. **장중 실행 (09:00~15:30):**
   - AEGIS 점수 실시간 모니터링
   - 규칙 기반 매매 집행

4. **장후 복기 (16:00):**
   - 당일 매매 기록 분석
   - 전략 성과 평가
   - 다음날 준비

### 위기 관리 프로토콜
1. **시스템 장애:**
   - 수동 백업 매매 절차 준비
   - 주요 포지션 핸드폰 알림 설정

2. **급락장 대응:**
   - 변동성 150% 이상 증가 시 모든 신규 진입 중지
   - 보유 포지션 손절폭 50% 축소

3. **연속 손실:**
   - 3연속 손실 시 1일 휴식
   - 5연속 손실 시 전략 재검토 의무화
